# BondStats Finora

**Version:** 1.0.0

BondStats Finora is a premium AI-powered Financial Intelligence Operating System.

Unlike traditional finance applications, Finora combines portfolio intelligence, financial research, macroeconomic learning, AI workspaces and investment analytics into one integrated platform.

# Architecture

```
src/

core/
layout/
components/
dashboard/
portfolio/
research/
learn/
future/
ai/
charts/
export/
utils/
design/

```

Every module follows the same architecture.

```
Model

↓

Controller

↓

View

↓

Renderer

↓

Storage

```

No module directly modifies another module.

Communication happens only through the Event Bus.
